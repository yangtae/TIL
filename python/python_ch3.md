## 2.문자열

- 문자열

    - 한줄로표현
  - 삼중 따옴표(""")
    - 여러 줄로 표현 가능
  - 여는 따옴표와 닫는 따옴표는 동일해야 함
  - 따옴표 안에 동일한 따옴표는 사용하지 못함



- 문자열 확장(문자 이스케이프)
